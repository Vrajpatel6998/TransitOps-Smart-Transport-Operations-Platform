# TransitOps - Smart Transport Operations Platform

TransitOps is a full-stack smart transport operations platform designed for managing fleet logistics, drivers, vehicle maintenance, fuel usage, and operational expenses.

## Repository Structure

The project is structured into the following directories:
- **`Backend/`**: Node.js, Express, TypeScript, and TypeORM backend connecting to a PostgreSQL database.
- **`Frontend/`**: Next.js App Router frontend built with TailwindCSS and TypeScript.

For instructions on configuring, running, and deploying each service, refer to:
- [Backend README](./Backend/README.md) (coming soon / initialized inside directory)
- [Frontend README](./Frontend/README.md)
